O que seria Html:


Html é uma linguagem de marcação de iper texto, é focado no conteudo do site. Fazendo pareceria com CSS e o JavaScript para criar um site

CSS:

CSS (Cascading Style Sheet, ou Folha de Estilo em Cascatas) e a parte que tem foco no Design de um site, o Html cria o conteudo, e o Css faz ele ficara agradavel aos olhos, ninguem acessa um site bagunsado e feio!


JavaScript

JavaScript é realmente uma linguagem de programação, que é bem eficaz para criação de site, ela é focada na interação, Vidios, Menu, Atualização do site entre outras funções


<Tags>

O que são tags? as tags são oque nós utilisamos para criar o nosso site, como paragrafo, negrito, titulo, usamos tegs para colocar imagem, a URL do nosso site, o icone, links e diversos outras funções possiveis com tags. Quase todas as tags tem abertura e fechadura, como por exemplo a tag <head> <head / >
Outras tags não possuem fechadura como por exemplo a tag <hr>

lembrando que elas não estão sozinhas.
Para a criação de um site temos os parametros e as tags

Marcação de Texto

Para marcar um treixo de seu texo para melhor atenção e interpretação dos leitores, você precissara colocar a tag <mark>
Podemos mudar a cor do fundo usando o parametro style="background-color: ;"


Tamanho de Texto

Pdemos mudar o tamanho do nosso textos usando duas tags. Uma tag para um texto grande e outra para um texto pequeno
Usando a tag <big> temos um texto grande, e usando a tag <small> temos um texto pequeno
Porem a tag BIG não é considerada, faz mais sentido usar o CSS para mudar o tamanho da fonte de seu texto. Já a tag SMALL é considerada pois em contratos está presente lettras pequenas, possue sentido ela star ali


Texto Deletado/Texto inserido

Podemos Marcar um texto como excluido para indicar que ele deve ser lido mas não considerado.
Usando a tag <del>

Podemos marcar um texto como inserido para dar ènfase e indicar que ele foi colocado depois.
Usando a tag <ins>


Subrescrito/Subscrito

A formula da àgua é H2O não H2O! para não cometermos esse erro usamos a tag <2> para se tornar um numero Subrescrito, mas podemos usar com qualquer outro numero, H2O foi só um exemplo

Já para fazer um numero elevado a outro como 7 - x8 = 41 tornando o numero subscrito, suamos a tag <8>

Como Colocar uma Imagem em Seu Site

Abaixo voce vera uma imagem que está na mesma pasta.
Utilizando o código <src="nome da imagem" alt="nome de preferencia" >

podemos gerar imagem apartir de uma sub-pasta.
Utilizando o código <img src="pasta/nome da imagem" alt="nome de preferencia">

alem disto podemos gerar uma imagem apartir do endereço da imagem(URL).
Utilizando o codigo < img src="URL da imagem" alt="nome de preferencia">
